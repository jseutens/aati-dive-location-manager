== Description ==

AATI Dive Location Manager is a plugin that allows you to manage the dive locations of your diving center. With this plugin, you can easily add, edit, and delete dive locations, and display them on your website.

== Features ==

- Add, edit, and delete dive locations
- Display dive locations on your website
- Manage details of each dive location, such as name, description, coordinates, and more

== Installation ==

1. Upload the `aatidlm` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Dive Locations' menu in the WordPress admin area to manage your dive locations.

== Frequently Asked Questions ==

= Can I customize the display of dive locations on my website? =

Yes, you can customize the display of dive locations by using the available templates and shortcodes provided by the plugin.
